# Açelya Lejeune Portfolio

A multilingual portfolio website showcasing my work across UX/UI design, visual design, frontend development, and AI-powered applications.

🌐 **Live website:** [acelyalejeune.com](https://acelyalejeune.com)

---

## About the Portfolio

This portfolio brings together my experience as a UX/UI and visual designer with my growing focus on frontend development and AI application development.

It presents selected projects, case studies, technical experiments, and professional certifications in a responsive bilingual interface available in English and French.

The goal of the website is not only to display finished work, but also to demonstrate how I approach design systems, user experience, frontend architecture, and product development.
