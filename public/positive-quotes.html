Positive Quotes App

Positive Quotes App is the TFE project I created during the final year of my Frontend Developer programme. It is an Angular application built with TypeScript, Angular Material, SCSS and JSON Server, designed to help users discover, save and revisit motivational quotes.

The application includes registration and login, quote categories, favorites, quote history, account settings, dark and light themes, and dynamic background images loaded from an API. User data and interactions are saved through JSON Server, using a JSON file as a lightweight local data source.

Technologies & Tools Used

Creative & UI Tools

Created the logo and simple visual identity elements.

Prepared and adjusted visual assets and background images.

Planned the interface, screens and user flow.

Frontend & Data

Built the application structure with Angular and Angular Material.

Implemented application logic, state handling and typed components.

Styled the interface with SCSS, including dark and light theme layouts.

Used JSON Server to store users, favorites and quote history locally.

Creation Process

Positive Quotes App was developed as a final-year frontend project with a focus on building a complete Angular application: interface design, component structure, user actions, data handling and responsive visual presentation.

Concept & UI Direction

Defined the visual direction around calm, positive and nature-inspired imagery, with dark and light theme variations to give users a personalized experience.

Prototype & User Flow

Used Figma to plan key screens such as register, login, home, favorites, history and account settings before implementation.

Angular Implementation

Built the app with Angular, Angular Material and TypeScript, organizing the interface into reusable components and pages.

Data & Interaction Logic

Used JSON Server to simulate backend data storage for users, favorites and quote history, while external APIs were used for quotes and dynamic background images.

Features

Positive Quotes App includes practical frontend application features: authentication flow, quote categories, favorites, history, theme switching, account settings and dynamic background images.

Register & Login

Users can create an account and log in before using the app, allowing their favorites and history to be connected to their profile.

Quote Categories & API Data

Quotes can be browsed by category, with quote content loaded from an API and displayed in a clean, focused interface.

Favorites

Users can save favorite quotes and revisit them later from a dedicated favorites page.

Quote History

The app keeps track of viewed quotes, allowing users to return to quotes they have already explored.

Dark & Light Themes

Users can switch between dark and light themes, changing the visual atmosphere of the application.

Dynamic Background Images

Background images change through an API when pages are refreshed, giving the interface a more atmospheric and varied experience.
