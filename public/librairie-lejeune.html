Librairie Lejeune

Librairie Lejeune is a responsive eCommerce bookstore project created to present and manage a catalog of books, stationery and gift items. The project combines visual direction, UI layout, frontend integration and basic backend functionality.

The website includes public pages for visitors and an admin panel for managing products, categories and catalog content. It was built with HTML5, CSS3, SCSS, JavaScript, PHP and MySQL, with a Figma prototype used to plan the interface and user flow.

Creation Process

To build Librairie Lejeune, I combined visual direction, UI prototyping and web development. The goal was to create a clear online bookstore interface with practical catalog management features.

Logo & Visual Direction

Created the logo and visual direction with Adobe Illustrator, focusing on a simple and recognizable identity for the bookstore.

Prototype & UI Layout

Used Figma to plan the page structure, product browsing flow and interface layout before implementation.

Visual Assets

Prepared and adjusted visual assets with Adobe Photoshop to support the look and feel of the eCommerce interface.

Frontend & Backend Implementation

Implemented the website with HTML5, CSS3, SCSS, JavaScript, PHP and MySQL, including responsive pages, product management and dynamic catalog features.

Features

Librairie Lejeune includes practical eCommerce features for visitors and administrators: product catalog browsing, category management, search and filtering, responsive pages and an admin panel for catalog updates.

Admin Panel

The admin panel allows products to be added, edited, displayed or deleted, making catalog management easier.

Product Category Management

Product categories are managed with MySQL, allowing administrators to organize items and create a clearer browsing experience for visitors.

Responsive Interface

The website adapts to desktop, tablet and mobile screens to provide a consistent browsing experience.

Search & Filtering

Visitors can find products more easily using search and filtering options such as category, price range and keyword.
